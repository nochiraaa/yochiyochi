## Git で個人的に思い入れのあるコマンド

`git add -p`…変更を対話形式でインデックスに登録できます。１つのファイルで同時に複数の変更を行った際に、各変更を別コミットに積みたい時などに便利です。

`git rebase -i`…過去のコミットの結合、コミットメッセージの修正、コミットの削除などを行えます。

上記２つはよく使っています。
コマンドというよりオプションに対する思い入れが強いかも知れません。
